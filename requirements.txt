# NYC Crashes Data Analysis - Dependencies
#

# - pandas: Data processing and manipulation

# - requests: HTTP library for NYC Open Data API calls
# - python-dateutil: Date parsing and timezone handling
#
# Installation:
#   pip install -r requirements.txt
#
# Usage:
#   CLI: python3 main.py
#   Web: python3 run-web_app.py (opens http://localhost:5001)
#
# Note: tkinter (for Tkinter GUI in ui_tk.py) is part of Python stdlib,
#       no pip install needed. On Debian/Ubuntu: sudo apt install python3-tk
