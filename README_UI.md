UI quick start

Run the simple interactive CLI:

    python ui.py

Options:
- Type a file path or URL (or "sample" to use the included `data/sample_2024.csv`).
- Enter date ranges in YYYY-MM-DD form.
- Use menu to export report, filtered CSV, plots (requires `matplotlib`/`pandas`), or a folium heatmap (requires `folium`).

Dependencies (optional):
- pandas
- matplotlib
- folium

Install any optional libraries via `pip install pandas matplotlib folium` if you want plots/heatmaps.
